This is a Multimodal Retrieval-Augmented Generation (RAG) agent — a system that answers queries by first retrieving relevant information (from a 50k-document corpus), then generating a grounded response. What makes it "multimodal" is that the retrieval layer understands both images and text simultaneously using CLIP embeddings (a model from OpenAI that maps images and text into the same vector space). A query like "show me red sneakers" retrieves both text descriptions and actual product images ranked by semantic similarity.

The agent uses function calling to intelligently route: a vision-heavy query goes to the image retriever, a text-heavy query goes to the text retriever, and mixed queries hit both. Results are served via a FastAPI app running in Docker on AWS EC2, with Redis caching hot queries and PostgreSQL storing document metadata.
